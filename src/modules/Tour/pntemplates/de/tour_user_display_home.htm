Direkteinstieg

Der Internetauftritt kann über den Link Administration verwaltet werden (in der Regel befindet sich dieser in der oberen linken Ecke). Nach einer Abmeldung ist dieser Link wieder verschwunden. Anschließend kann man sich über den Login-Block (in der Regel auf der rechten Seite) oder über die Login-Seite erneut anmelden.

Derzeit liegt keine Anmeldung vor. Mit den, während der Installation festgelegten, Benutzerdaten kann über den Login-Block (normalerweise auf der rechten Seite) oder alternativ über die Login-Seite eine Anmeldung durchgeführt werden. Anschließend erscheint im Hauptmenü (links oben) der Link zur Administration.

Später, um dieses Tour-Modul zu entfernen und mit der Erstellung der Website zu beginnen...

...begibt man sich einfach in die Administration, klickt auf die Registerkarte "System" und öffnet das Modul "Einstellungen". Um ein anderes Modul auf der Startseite anzuzeigen, ändern man die Auswahl im Abschnitt "Startseite" auf ein beliebiges Modul.
